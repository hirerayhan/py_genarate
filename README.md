তোমার পাইথন স্ক্রিপ্টকে `.exe` অ্যাপে কনভার্ট করতে এবং ডিকোড (Reverse Engineering) থেকে বাঁচাতে `PyArmor` (কোড এনক্রিপ্ট করার জন্য) এবং `PyInstaller` (অ্যাপ বানানোর জন্য) ব্যবহার করতে হবে।

CMD ওপেন করে ধাপে ধাপে এই কমান্ডগুলো দাও:

১. প্রথমে প্রয়োজনীয় টুলগুলো ইনস্টল করো:
```cmd
pip install pyinstaller pyarmor
```

২. এবার তোমার কোডটাকে এনক্রিপ্ট/লক করো (যাতে কেউ ডিকোড করতে না পারে):
```cmd
pyarmor gen generate_numbers.py
```
*(এই কমান্ড দিলে `dist` নামে একটা ফোল্ডার তৈরি হবে, যার ভেতর তোমার লক করা কোড থাকবে)*

৩. এবার ওই `dist` ফোল্ডারের ভেতরে প্রবেশ করো:
```cmd
cd dist
```

৪. সবশেষে তোমার নামে অ্যাপ (exe) তৈরি করো:
```cmd
pyinstaller --onefile --name "Rayhan_Biswas_Tool" generate_numbers.py
```

**অ্যাপ কোথায় পাবে?**
কাজ শেষ হলে দেখবে বর্তমান `dist` ফোল্ডারের ভেতরে আরও একটি নতুন `dist` ফোল্ডার তৈরি হয়েছে। ওই ফোল্ডারের ভেতরেই তোমার `Rayhan_Biswas_Tool.exe` ফাইলটি পেয়ে যাবে। 

এখন এই অ্যাপটা তুমি নিশ্চিন্তে ব্যবহার করতে পারো বা অন্য কাউকে দিতে পারো, কেউ ডিকোড করে তোমার মূল কোড বের করতে পারবে না।
